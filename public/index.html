Cateroids

Game Design Document

By Zheng Lu, Alex Espinal, Kyle Simon



Introduction

This document describes a game called “Cateroids”, which is a 2D run and gun side scroller game. The game will employ sprite-based animations, physics, AI, side scrolling, and multiplayer.

 

Technology

This game will be developed using Phaser 3, a javascript game engine that uses WebGL to create browser games.

 

Back Story

It is a period of uncivil war. Feline spaceships, pouncing from a hidden base, have won the first victory against the evil Galactic Pawpire. During the battle, Feline spies managed to steal secret plans to the Pawpire’s ultimate weapon, the WOLF STAR, an armored space station with enough power to destroy the entire feline race.

 

Leonardo DaCatmeow, races to his home LitterBox-380.01 in the Alpha-Catauri Quadrant with the stolen plans in hand that can save his people….

 

Objective

Leonardo DaCatmeow’s goal is to find his way back home to deliver the stolen plans. But through his travels, the Galactic Pawpire has sent out troops to stop Leonardo. Leonardo must traverse his way through an asteroid filled space while destroying his enemies.

 

The Galactic Pawpire has to take back the stolen plans or they face extinction. They will do whatever it takes to get it back. However, resources are limited and the Galactic Pawpire must strategize on how it will send out their limited amount of troops to defeat Leonardo.

 

Gameplay

The game will work like a side scroller. Leonardo will be able to move freely in space in all directions but the screen will only scroll right. Leonardo will be able to shoot enemies and use his grappling hook to attach to asteroids and avoid enemy attacks. His ship has been damaged after stealing the secret plans and his oxygen depletes over time. He must land on oxygen filled asteroids to replenish his oxygen.




Player 1 Controls

  • Moving the mouse - change the direction of where the cat aims
  • Left click - shoot gun
  • WASD - move the cat
  • 1 - 0  - switch to different weapons
  • Shift - cast and reel grappling hook

 

Player 2 Controls

  • Mouse Click - choose which enemies to place on the field and choose a position to place them

User Interface

  • Splash Screen - includes the game logo and start button to take the user to the start screen. Hype music will play in the background.
  • Start Screen Menu- The start screen menu will come after the splash screen. It will have the following options:
    • Start new game - takes the user into the multiplayer selection screen
    • Controls - show controls for both players
    • About - show information about the game authors
  • Multiplayer Selection Screen - this is where the players decide on which player they want to be. After choosing the characters, the game will start after pressing the start game button. The choices will be:
    • Cat - the main protagonist of the story
    • Dog - the main antagonist of the story
    • Start Game - only works after both players chooses one player
    • Back - return to the start screen menu
  • In Game GUI - the following will be displayed at all times:
    • Player 1
      • health meter
      • oxygen meter
      • ammo counter
      • different weapons
      • currency counter
    • Player 2
      • Troop counter
      • Attack counter
  • Game over - when the main protagonist loses all its health. The following choices will appear
    • Restart - takes both players back to the first level
    • Exit - exit the game in-progress and takes the user to the splash page
  • End screen - when the main protagonist completes the game. A thank you for playing message will show and credits will roll.
  • Pause screen - when the user presses the ESC key, a pause menu will appear which will freeze all movement in the game. The pause screen will have the following options:
    • Resume game - continue playing the game
    • Controls - show a diagram of the controls
    • Exit game - exit the game and take the user to the splash screen

 

Game Mechanics

  • Player 1
    • Asteroids and enemies spawn randomly
    • Oxygen decreases over time
    • If oxygen is low - the player must land on a special type of asteroid and defend from waves of enemies while oxygen is being replenished.
    • If hit by an enemy or asteroid - reduce health
    • Use grappling hook to increase velocity and to land on asteroids. After a certain amount of time, the player slows down and goes back to normal
  • Player 2
    • Spawn in troops
    • Trigger events and attacks

 

Assets

  • Art Work: all artwork in the game will be original.
  • Sprites:
    • Main character - cat inside a spaceship. We will need the animations for:
      • Moving - flipping and rotating will be used to mimic different movements
      • Attacking
      • Taking damage
      • Dying
    • Asteroids - there will be 4 types of asteroids: small, medium, large, and the oxygen asteroid. Each will need the following animations:
      • Breaking
    • Enemies - dog aliens. Different breeds will have different behaviors/abilities. Each alien will need the following animations:
      • Moving
      • Attacking
      • Dying
    • Boss - the boss at the end of the game. We will need the animations for:
      • Attacking
      • Taking damage
      • Dying
    • Weapons - The following weapons will be used by the main protagonist:
      • Gun - shoots a basic ball of energy
    • Background - The background artwork that will appear behind all the levels
  • Music: all music and sound effects will be original.
  • Sound Effects:
    • Shooting
    • Enemy attack
    • Damage taken
    • Oxygen completely refilled
